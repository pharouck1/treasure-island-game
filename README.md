**Treasure Island Game**

Welcome to Treasure Island, a simple text-based adventure game built with Python.

**Game Description**

Your mission is to find the hidden treasure by making the right choices.

Choose your path carefully
Each decision affects the outcome
One wrong move… and it's Game Over 
**Features**
Interactive command-line gameplay
Multiple decision paths
Different endings based on user choices

*  User input handling
Basic game logic
